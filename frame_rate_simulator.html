Scenario presets

Narrow Sector Width

더 얇은 wedge는 필요한 scan line 수를 줄입니다. 같은 깊이와 focus 조건이라면 가장 즉각적으로 frame rate를 끌어올리는 조정입니다.

Decrease Imaging Depth

초음파는 더 깊은 구조에서 echo가 돌아오기를 더 오래 기다려야 합니다. depth를 줄이면 line마다 대기 시간이 짧아져 frame 생성이 빨라집니다.

Reduce Focal Zones

focus를 여러 개 두면 한 frame을 만들기 위해 같은 line을 반복 전송해야 합니다. single focus는 temporal resolution을 위해 가장 강력한 타협입니다.

This simulator uses a line-by-line educational model: PRFmax ≈ c / (2D), frame rate ≈ PRF / (lines per frame × focal zones), and lines per frame ≈ sector width × line density. It is intended to stay close to standard teaching physics, not to reproduce a specific vendor pipeline exactly.
